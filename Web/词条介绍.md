# 📘 词条介绍
本页用于介绍软件中各类名词与功能的含义说明。

---

## 🌐 全局操作
全局操作用于控制软件整体运行状态：

| 功能 | 说明 |
|------|------|
| 🌙 **休眠** | 立即终止所有宏，并禁用软件全部功能；取消休眠后恢复运行。 |
| ⏸️ **暂停** | 暂停当前所有宏，再次点击继续执行。 |
| ⛔ **终止所有宏** | 强制停止所有正在运行的宏任务，**不可恢复**，用于紧急情况。 |
| 🔄 **重载** | 重新加载当前配置，撤销未保存的修改。 |
| 💾 **应用并保存** | 保存并立即应用所有修改，未点击前不会生效。 |

## 🗂️ 页签说明
软件主界面通过不同页签对功能进行分类管理：

| 页签 | 说明 |
|------|------|
| ⌨️ **按键宏** | 通过快捷键触发宏，支持单键与组合键（如 `Ctrl + C`）。 |
| 🔤 **字串宏** | 通过输入特定字符序列触发宏（如连续按键、输入字符串）。 |
| 🧭 **菜单宏** | 通过快捷键呼出菜单轮，再由菜单按钮触发对应宏。 |
| ⏰ **定时宏** | 按时间触发宏，支持间隔、每日、每周、每月等方式。 |
| 🧩 **宏模块** | 可被其他宏调用的宏片段，支持嵌套与变量继承。 |
| 🎮 **按键替换** | 将某个按键映射为其他按键或组合键。 |
| 🛠️ **工具** | 辅助功能集合，如截图、录制、取色、窗口信息等。 |
| ⚙️ **设置** | 修改软件的全局配置与行为选项。 |
| ❓ **帮助** | 使用说明、开源地址、交流群、反馈与更新日志。 |
| ❤️ **打赏作者** | 支持作者持续维护与功能迭代。 |
| 🌟 **特别感谢** | 感谢代码贡献者、开源项目与社区成员。 |

## ⚙️ 宏配置
用于定义单条宏的触发方式与执行行为：

| 参数 | 说明 |
|------|------|
| 🎯 **触发键** | 设置宏的触发按键与方式：<br> - **按下**：按下即触发，未松开可能重复触发<br> - **松开**：松开时触发<br> - **松止**：按下启动，松开停止<br> - **开关**：首次启动，再次停止<br> - **长按**：持续按下到指定时间后触发 |
| 📜 **宏指令** | 定义宏的执行指令流程。 |
| 🎛️ **触发键编辑** | 打开触发键设置界面，进行可视化配置。 |
| 🗂️ **宏指令编辑** | 打开宏指令编辑器，编辑执行内容。 |
| 🎮 **按键模拟方式** | - **AHK Send**：通用方式，适合办公软件与大多数游戏（管理员权限可以让更多游戏有效）<br> - **keybd_event**：系统接口方式，适合老软件或游戏（需管理员权限）<br> - **罗技驱动**：调用 Logitech 驱动模拟按键（需管理员权限且 GHUB 已运行）<br> ⚠️ *keybd_event 与 罗技按键可作为触发键，切勿触发自身以避免死循环* |
| 🚫 **禁止宏** | 禁用当前宏配置，不参与触发与执行。 |
| 🗑️ **删除宏** | 删除当前宏配置（不可恢复）。 |
| 🔁 **循环次数** | 设置宏的重复执行次数。 |
| 🪟 **前台限制** | 仅在指定前台窗口生效（留空表示全局）。 |
| 📝 **备注** | 对宏用途进行说明，便于管理与维护。 |

## ⚙️ 软件设置
用于调节整体行为与模拟真实性的全局参数：

| 设置项 | 说明 |
|--------|------|
| ⏱️ **按住时间浮动（%）** | 📊 模拟按键持续时间的浮动范围。例如：设为 10，原 50ms → 最终 45~55ms。 | 
| ⏳ **每次间隔时间浮动（%）** | 📉 每个按键动作间隔的浮动范围。例如：设为 10，原 500ms → 实际 450~550ms。 | 
| ⌛ **间隔指令时间浮动（%）** | 📈 每组指令之间的等待时间浮动范围。例如：设为 10，原 3000ms → 实际 2700~3300ms。 | 
| 📐 **坐标 X 浮动（px）** | ↔️ 模拟点击时 X 坐标的浮动范围。例如：设为 30，原 X=900 → 实际 870~930。 | 
| 📐 **坐标 Y 浮动（px）** | ↕️ 模拟点击时 Y 坐标的浮动范围。例如：设为 50，原 Y=400 → 实际 350~450。 | 
| 🧵 **多线程数（0~10）** | 🌀 大于 0 时启用多线程。 | 
| ⚡ **开机自动启动** | 💻 开机时自动运行软件。 | 
| 👀 **指令显示** | 📺 执行指令时在窗口中显示。 |
| 🚨 **无变量提醒** | 使用不存在变量时弹窗提示。 |
| 📍 **菜单轮位置固定** | 菜单轮呼出时保持固定位置。 |
| ── **分割线显示** | 在宏配置之间显示分割线。 |
| 🌐 **语言 / Lang** | 设置软件显示语言。 |
| 🔤 **软件字体** | 设置界面字体。 |
| 📸 **截图方式** | 设置截图实现方式。 |
