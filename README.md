## 🧩 项目介绍

**RMT（若梦兔）** 是一款基于 **AutoHotkey v2** 的免费开源按键宏工具，  
致力于让用户通过简单配置即可实现各类自动化操作。  

✨ 支持功能包括：
- ⌨️ **按键宏 / 鼠标宏 / 手柄宏** 的录制与播放  
- 🔍 **图片识别、颜色识别、文本识别** 等视觉自动化能力  
- ⚙️ **逻辑判断、流程控制、多线程执行**  
- 💡 可广泛应用于游戏辅助、办公自动化、脚本执行等场景  

> 💬 无论是重复操作还是复杂交互流程，RMT 都能让你的电脑「自动工作」。
